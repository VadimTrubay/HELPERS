curl (Client URL) — это командная утилита, используемая для отправки HTTP-запросов и взаимодействия с веб-серверами. Она поддерживает множество различных протоколов, включая HTTP, HTTPS, FTP и другие. curl часто используется разработчиками и системными администраторами для тестирования API, загрузки файлов или автоматизации сетевых запросов.

Основные возможности curl
Отправка HTTP-запросов (GET, POST, PUT, DELETE и другие).
Загрузка файлов с удалённых серверов.
Передача данных с использованием различных протоколов (HTTP, HTTPS, FTP, SCP, и др.).
Поддержка аутентификации, SSL/TLS, прокси-серверов и cookies.
Основной синтаксис
bash
Копіювати код
curl [опции] [URL]
Примеры использования curl
Простой GET запрос:

bash
Копіювати код
curl https://api.example.com
Отправляет простой GET запрос на указанный URL и выводит ответ.

GET запрос с заголовками:

bash
Копіювати код
curl -H "Accept: application/json" https://api.example.com
Добавляет заголовок Accept: application/json к запросу.

POST запрос с данными:

bash
Копіювати код
curl -X POST -d "name=John&age=30" https://api.example.com/users
Отправляет POST запрос с данными формы на сервер.

POST запрос с JSON:

bash
Копіювати код
curl -X POST -H "Content-Type: application/json" -d '{"name": "John", "age": 30}' https://api.example.com/users
Отправляет POST запрос с JSON-данными. Это удобно при работе с API.

Сохранение ответа в файл:

bash
Копіювати код
curl -o output.html https://example.com
Сохраняет ответ сервера в файл output.html.

Проверка заголовков ответа:

bash
Копіювати код
curl -I https://example.com
Отправляет запрос и выводит только заголовки ответа.

Проверка статуса сервера:

bash
Копіювати код
curl -o /dev/null -s -w "%{http_code}\n" https://example.com
Выводит только HTTP-код статуса ответа (например, 200 или 404).

Аутентификация:

bash
Копіювати код
curl -u username:password https://api.example.com/secure-data
Отправляет запрос с использованием базовой аутентификации.

Полезные опции curl
-X — Указание метода запроса (GET, POST, PUT, DELETE и т.д.).
-H — Добавление заголовков к запросу.
-d — Передача данных (для POST или PUT запросов).
-o — Сохранение ответа в файл.
-I — Отправка запроса и возврат только заголовков ответа.
-u — Указание имени пользователя и пароля для аутентификации.
curl — мощный инструмент, который можно использовать для тестирования и отладки веб-приложений, работы с API и автоматизации сетевых операций.
